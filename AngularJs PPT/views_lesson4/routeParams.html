AngularJS 的路由机制 -- 参数传递 $routeParams

另外一个场景：在用户切换到另外一个route时，需要传递当前页面的数据到下一个页面，这个在AngularJS中如何实现呢？

$routeParams

$routeParams服务允许我们获取当前页的路由参数(route parameters )集合。

	.when('/router/:id', {
			templateUrl : 'router.html',
			controller  : 'routerCtrl'
	})
	<a href="#/router/10">link</a>
	app.controller("routerCtrl", ['$scope', '$routeParams',function($scope, $routeParams) {
			console.log($routeParams.id); //  10 
	}]);

本质

路由参数其实是$location服务的search()方法和path()方法返回结果的集合，当前路径跟路由路径相匹配的时候提取。

在参数名称冲突的情况下，path结果优先于search结果。

		//URL -- http://example.com/#/some/path?foo=bar&baz=xoxo
		var searchObject = $location.search();
		//Result --  {foo: 'bar', baz: 'xoxo'}

$location服务保证$routeParams对象的标识在发生$route变化时保持不变（但其仍有可能会变化）。

注意

$routeParams只有在路径改变成功完成后才更新。这意味着$routeParams并不总能正确的获取结果。

这个时候你可以使用$ route.current.params访问新的参数。（注意他们变化的顺序。。。）
